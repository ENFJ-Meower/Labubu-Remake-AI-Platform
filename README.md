# Labubu Remake AI Platform

## 项目概述
Labubu Remake AI Platform 是一个现代化的AI智能体平台，提供工作流设计、管理和执行的完整解决方案。

## 核心功能

### 1. AI Agent 工作流设计
- **可视化工作流编辑器** - 拖拽式节点编辑，支持多种AI服务节点
- **实时连接系统** - 智能连接线，支持缩放和画布移动
- **节点配置面板** - 完整的节点参数配置界面
- **工作流验证** - 自动检测工作流完整性和有效性

### 2. 工作流管理系统 ✨ 新增功能
- **工作流列表管理** - 查看和管理所有DAG工作流
- **统计面板** - 实时显示工作流状态统计
- **搜索和筛选** - 支持按状态、名称筛选工作流
- **工作流操作** - 查看、复制、删除工作流等操作

### 3. 实时状态监控 ✨ 新增功能
- **状态监控面板** - 实时显示工作流和节点执行状态
- **SSE连接优化** - 自动重连机制，连接状态指示器
- **节点进度跟踪** - 实时显示每个节点的执行进度
- **执行日志查看** - 完整的执行日志记录和展示

### 4. 执行结果查看器 ✨ 新增功能
- **结果概览** - 统计成功、失败消息数量
- **消息列表** - 详细的输入输出消息记录
- **结果筛选** - 按消息类型筛选查看
- **结果导出** - 支持JSON格式导出执行结果

### 5. 错误处理与恢复 ✨ 新增功能
- **全局错误处理** - 统一的错误提示和处理机制
- **自动重试机制** - 网络错误自动重连
- **错误日志记录** - 本地存储错误日志，便于调试
- **用户友好提示** - 清晰的错误信息和解决建议

## 技术架构

### 前端技术栈
- **Vue.js 3** - 响应式前端框架
- **组件化设计** - 模块化组件架构
- **CSS3 动画** - 流畅的交互动画效果
- **响应式布局** - 支持移动端和桌面端

### API 集成
- **RESTful API** - 标准的REST接口调用
- **Server-Sent Events** - 实时数据推送
- **错误重试机制** - 自动处理网络异常
- **批量数据处理** - 优化性能的数据更新机制

### 数据管理
- **状态管理** - 集中的应用状态管理
- **本地存储** - 错误日志和用户配置存储
- **数据缓存** - 提高应用响应速度
- **数据验证** - 确保数据完整性和有效性

## 使用指南

### 工作流设计
1. 在"Conversation Flow"标签页中设计工作流
2. 从左侧面板拖拽节点到画布
3. 连接节点创建工作流逻辑
4. 配置节点参数和设置
5. 保存并测试工作流

### 工作流管理
1. 切换到"工作流管理"标签页
2. 查看所有工作流的状态统计
3. 使用搜索和筛选功能快速找到目标工作流
4. 点击工作流卡片查看详情
5. 使用操作按钮进行管理操作

### 状态监控
1. 在工作流列表中点击"📊"按钮
2. 查看工作流的实时执行状态
3. 监控各节点的执行进度
4. 查看执行日志和错误信息
5. 使用刷新按钮更新状态

### 结果查看
1. 在工作流列表中点击"📋"按钮
2. 查看执行结果统计
3. 浏览详细的消息记录
4. 使用筛选功能查看特定类型消息
5. 导出结果用于进一步分析
